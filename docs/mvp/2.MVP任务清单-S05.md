### User Stories

#### US-5.1 文本编辑功能
**作为** 用户
**我想要** 编辑转写结果
**以便** 修正错误

**验收标准**:
- [x] 可以编辑文本
- [x] 自动保存
- [x] 保留时间戳

**任务拆解**:
- [x] Task-5.1.1: 实现文本编辑器组件（4h）完成时间：2025-11-21
- [x] Task-5.1.2: 实现自动保存（2h）完成时间：2025-11-21
- [x] Task-5.1.3: 实现更新 API（2h）完成时间：2025-11-21
  - PUT /api/projects/:id/transcription
- [x] Task-5.1.4: 添加撤销/重做功能（2h）完成时间：2025-11-21
- [x] Task-5.1.5: 扩展时间轴列宽与超长时间戳适配（2h）完成时间：2025-11-21
  - 以 `999 min 00.0 s` 为基准重构栅格宽度，并对窄屏做响应式处理
- [x] Task-5.1.6: 文本区信息头与过滤控件优化（3h）完成时间：2025-11-21
