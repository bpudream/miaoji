
# Sprint 04: 翻译体验 (前端交互)

**Sprint 目标**：
在前端实现双语对照视图，允许用户触发翻译、查看翻译进度，并对翻译结果进行校对和导出。

**周期**：2026-02-27 ~ 2026-03-06
**状态**：已实现 (Implemented)

---

## User Stories

### US-4.1 翻译触发与状态管理

**作为** 用户
**我想要** 在界面上选择目标语言并开始翻译
**以便** 系统开始后台处理。

**验收标准**:

* [x] 详情页新增“AI 翻译”按钮，点击弹出语言选择模态框。
* [x] 提交后显示翻译进度条（或 Loading 状态）。
* [x] 支持轮询更新任务状态。

**任务拆解**:

* [x] Task-4.1.1: 实现翻译设置模态框 (选择语言)。（估时：2h）(完成时间：2026-02-06)
* [x] Task-4.1.2: 对接后端 `/api/projects/:id/translate` 与 `/api/projects/:id/translations`。（估时：1.5h）(完成时间：2026-02-06)

---

### US-4.2 双语对照视图 (Bilingual View)

**作为** 用户
**我想要** 并排对照查看原文和译文
**以便** 我可以理解内容并核对翻译质量。

**验收标准**:

* [x] 转写结果区域支持切换视图模式：仅原文 / 仅译文 / 双语对照。
* [x] 点击译文的时间戳，视频播放器能跳转到对应时间。
* [x] 译文与原文段落高度对齐（按 index 对齐）。

**任务拆解**:

* [x] Task-4.2.1: 基于 `TranscriptionPanel` 增加双语/译文视图组件。（估时：4h）(完成时间：2026-02-06)
* [x] Task-4.2.2: 实现视图切换逻辑（本地状态）。（估时：1h）(完成时间：2026-02-06)

---

### US-4.3 译文导出

**作为** 用户
**我想要** 导出翻译后的 SRT 字幕
**以便** 制作双语字幕视频。

**验收标准**:

* [x] 导出菜单增加“导出译文 SRT”和“导出双语 SRT”选项。
* [x] 双语 SRT 格式正确（原文在上，译文在下）。

**任务拆解**:

* [x] Task-4.3.1: 后端扩展导出接口，支持合并双语内容。（估时：2h）(完成时间：2026-02-06)
* [x] Task-4.3.2: 前端对接新的导出选项。（估时：0.5h）(完成时间：2026-02-06)

---
